# Nginx-Deploy-Configure-Microservices

- Framework: 
  - Back-end: Java Spring boot
  - Front-end: Vuejs

- Run project
  - Deploy: docker-compose up -d
  - Browser:
    - http://localhost/ => front-end-client
    - http://localhost/admin => front-end-admin
    - http://localhost/api => back-end
